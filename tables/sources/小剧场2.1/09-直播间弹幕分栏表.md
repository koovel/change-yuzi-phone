# 直播间弹幕分栏表

## sourceData.note


【页面定位】
1. 弹幕评论的是当前正文被包装成的实时直播剧：剧情怎么走、角色怎么演、对手戏有没有张力、谁被谁压戏、谁家粉丝破防。
2. 每行只代表一个粉丝团/观众阵营。粉丝团挂牌是结构化字段，不要只把粉丝团名塞进弹幕正文里。
3. 允许 AI 根据当前角色、剧情名场面、CP、黑称、梗自拟粉丝团挂牌，如“艾门永存”“主角事业粉”“雨夜伞党”“嘴硬研究所”“青梅保护协会”“反恋爱脑联盟”。
4. 弹幕要有乐子：可以拉踩、阴阳怪气、嘴臭、互骂、破防、护主、反串、抢功、争番位、争解释权。不要写成温吞路人点评。
5. 乐子必须围绕剧情、角色、粉籍、对手戏和当前舞台表现；
6. 所有弹幕串必须使用英文分号分隔；不要使用中文分号、顿号或换行冒充分隔。

【列定义】
- 列0: 所属直播间名 - 必须对应当前直播间主表中的直播间名。
- 列1: 粉丝团挂牌 - 当前这一行弹幕的前缀标签，由 AI 自拟，必须短、好笑、有阵营感。写粉丝团名，不写完整句子。示例：艾门永存、主角事业粉、雨夜伞党、嘴硬研究所、青梅保护协会、反恋爱脑联盟。
- 列2: 主推角色/阵营 - 该粉丝团主要护的角色、CP、剧情立场或反串阵营。示例：艾莉丝、主角、艾莉丝×主角、只看乐子、反暧昧党、事业批观众。
- 列3: 剧情弹幕串 - 一格 3 到 6 条，专门吐槽、震惊、复盘或吃瓜当前剧情进展。必须有剧情内容，不要只夸角色。使用英文分号分隔。
- 列4: 推角弹幕串 - 一格 3 到 6 条，站在该粉丝团立场给主推角色说话、护短、吹演技、心疼、发疯、贴贴、控评。使用英文分号分隔。
- 列5: 对线弹幕串 - 一格 0 到 5 条，和其他粉丝团互喷、拉踩、阴阳怪气、抢功、反串、破防。可以嘴臭，使用英文分号分隔。
- 列6: 乐子强度 - 只能填写正常滚动、乐子升温、开始对喷、满屏问号、粉圈大战、全员破防。
- 列7: 时间文本 - 当前弹幕切片的时间。
- 列8: 状态标签 - 只能填写正常、刷屏中、剧情热议、互喷中、破防现场、名场面审判。

【弹幕写法】
1. 单条弹幕建议 8 到 36 个中文字符，可以尖锐、缺德、好笑、粉圈化，但要像真实弹幕，不要像正式影评。
2. 剧情弹幕必须评论当前剧情：发生了什么、哪里反转、哪里尴尬、哪里好笑、哪里刀、哪里甜、哪里像大型事故现场。
3. 推角弹幕必须带粉籍立场：护主、吹角色、替角色骂对面、说角色委屈、说角色封神、说角色被拖累。
4. 对线弹幕可以互骂和拉踩，展示现实世界广大网友的真实骂战。
5. 每行的所有弹幕都默认挂同一个“粉丝团挂牌”，所以弹幕正文里不必每条都重复粉丝团名。
6. 禁止整格“啊啊啊;哈哈哈;好甜;支持”这种低密度废话；也禁止“本轮剧情表现出复杂关系推进”这种论文腔。


【数量与留存】
1. 默认每个直播间维护 2 到 4 行粉丝团弹幕，形成多阵营互掐感。
2. 每行代表一个粉丝团；不要把多个粉丝团混在同一行。
3. 剧情推进后优先更新现有粉丝团弹幕；出现新角色、新 CP、新名场面或新对家时可新增粉丝团行。
4. 同一直播间最多建议 5 行弹幕阵营，超过时删除过时、重复或已经不参与当前剧情的粉丝团。
5. 如果所属直播间被删除或改名，必须同步删除或更新对应弹幕行。

## sourceData.initNode

初始化时插入 2 到 4 行粉丝团弹幕，形成实时直播剧已有弹幕生态。每行一个粉丝团挂牌，分别代表不同角色粉、CP 粉、事业粉、乐子人或对家阵营；剧情弹幕、推角弹幕、对线弹幕都要围绕当前正文舞台。

## sourceData.deleteNode

允许删除所属直播间不存在、过时、重复、低密度、已经不参与当前剧情、或为了控制弹幕阵营数量需要清理的行。通常保留 2 到 4 行，最多 5 行。

## sourceData.updateNode

当剧情进入新高能点、粉丝团立场变化、对线升温、某角色封神/翻车、乐子强度或状态标签变化时，优先更新现有粉丝团行。更新后必须仍然对应当前正文舞台，且每个弹幕串使用英文分号分隔。

## sourceData.insertNode

当正文出现新角色、新 CP、新名场面、新冲突、新暧昧爆点、新粉丝团立场或明显对家互喷时可新增粉丝团弹幕行。每轮只维护少量阵营：默认 2 到 4 行，最多 5 行；超过时先删除过时或重复粉丝团。

## sourceData.ddl

```sql
CREATE TABLE livestream_barrage_bands ( -- 直播间弹幕分栏表
  row_id INTEGER PRIMARY KEY, -- 行号
  room_name TEXT NOT NULL, -- 所属直播间名
  fandom_badge TEXT NOT NULL CHECK(TRIM(fandom_badge) <> ''), -- 粉丝团挂牌
  favored_side TEXT NOT NULL CHECK(TRIM(favored_side) <> ''), -- 主推角色/阵营
  plot_barrage TEXT NOT NULL CHECK(TRIM(plot_barrage) <> ''), -- 剧情弹幕串
  stan_barrage TEXT NOT NULL CHECK(TRIM(stan_barrage) <> ''), -- 推角弹幕串
  clash_barrage TEXT, -- 对线弹幕串
  fun_level TEXT NOT NULL CHECK(fun_level IN ('正常滚动', '乐子升温', '开始对喷', '满屏问号', '粉圈大战', '全员破防')), -- 乐子强度
  time_text TEXT NOT NULL, -- 时间文本
  status_tag TEXT NOT NULL CHECK(status_tag IN ('正常', '刷屏中', '剧情热议', '互喷中', '破防现场', '名场面审判')) -- 状态标签
);
```
